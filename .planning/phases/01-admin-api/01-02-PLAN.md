**模型 CRUD 端点组：**

1. GET /api/admin/models — 列出所有模型
   - 调用 `get_all_models()`
   - 返回前对每个模型脱敏：`api_key` 只显示前 6 位 + "****"（如果 api_key 长度 > 6）
   - 返回 `{"models": [脱敏后的模型列表]}`

2. POST /api/admin/models — 创建模型
   - 接受 JSON body: `{"name": str, "api_url": str, "api_key": str, "model_id": str, "daily_token_limit": int, "daily_call_limit": int, "priority": int}`
   - 校验必填字段：name, api_url, api_key 非空
   - 调用 `create_model(model_data)`
   - 返回 `{"message": "创建成功", "config_id": config_id}`

3. PUT /api/admin/models/{config_id} — 更新模型
   - 接受 JSON body: 同创建，额外含 `is_default_model: int`
   - 调用 `update_model(config_id, model_data)`
   - 返回 `{"message": "更新成功"}`

4. DELETE /api/admin/models/{config_id} — 删除模型
   - 调用 `delete_model(config_id)`
   - 返回 `{"message": "删除成功"}`

5. PUT /api/admin/models/{config_id}/set-default — 设置默认模型
   - 先获取当前模型信息 `get_all_models()` 找到该 config_id 的模型
   - 调用 `update_model(config_id, {..., is_default_model: 1})` — update_model 内部会自动清除其他默认标记
   - 返回 `{"message": "默认模型已设置"}`

**API Key 管理端点组：**

6. GET /api/admin/keys — 列出所有 API Key
   - 调用 `get_all_api_keys()`
   - 返回 `{"keys": [keys]}`

7. POST /api/admin/keys — 创建 API Key
   - 接受 JSON body: `{"name": str}`
   - 校验 name 非空
   - 调用 `create_api_key(name)`
   - 返回 `{"message": "创建成功", "key": result}`（包含完整的 api_key，仅此一次）

8. DELETE /api/admin/keys/{key_id} — 删除（软删除）API Key
   - 调用 `delete_api_key(key_id)`
   - 返回 `{"message": "删除成功"}`

**统计端点组：**

9. GET /api/admin/stats/daily — 每日统计
   - 调用 `get_daily_stats()`
   - 对返回数据脱敏：api_url 保留，api_key 脱敏（同上）
   - 返回 `{"stats": [脱敏后的统计列表]}`

10. GET /api/admin/stats/call-logs — 调用日志
    - 查询参数：limit: int = Query(100, ge=1, le=1000), offset: int = Query(0, ge=0)
    - 调用 `get_call_logs(limit=limit, offset=offset)`
    - 返回 `{"logs": [logs], "limit": limit, "offset": offset}`

**系统配置端点：**

11. GET /api/admin/system/auto-switch — 获取自动切换状态
    - 调用 `get_auto_switch_status()`
    - 返回 `{"enabled": bool}`

12. PUT /api/admin/system/auto-switch — 设置自动切换
    - 接受 JSON body: `{"enabled": bool}`
    - 调用 `set_auto_switch_status(enabled)`
    - 返回 `{"message": "设置成功", "enabled": enabled}`
