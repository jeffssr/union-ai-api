在 app/main.py 中做以下修改：

1. 在 import 区域添加：
```python
from fastapi.staticfiles import StaticFiles
from fastapi.responses import FileResponse
import os
```

2. 在 `@app.get("/")` 路由之后，文件末尾之前，添加静态文件服务逻辑。

   **关键：必须按照以下顺序添加，这是匹配优先级的关键。**

   先删除现有的 `@app.get("/")` 根路由（它会被 catch-all 替代）。

   然后按顺序添加：

   a) 挂载 /assets 静态资源（React 构建产物的 JS/CSS/图片）：
   ```python
   # 静态文件服务（React 构建产物）
   STATIC_DIR = os.path.join(os.path.dirname(__file__), "static")

   if os.path.exists(os.path.join(STATIC_DIR, "assets")):
       app.mount("/assets", StaticFiles(directory=os.path.join(STATIC_DIR, "assets")), name="static_assets")
   ```

   b) SPA catch-all 路由（必须最后注册，最低优先级）：
   ```python
   @app.get("/{full_path:path}")
   async def serve_spa(full_path: str):
       """SPA fallback: 未匹配的路径返回 index.html，由 React Router 处理"""
       index_path = os.path.join(STATIC_DIR, "index.html")
       if os.path.exists(index_path):
           return FileResponse(index_path, media_type="text/html")
       # 无 index.html 时返回 API 状态（向后兼容纯 API 模式）
       return {"message": "AI Proxy API is running"}
   ```

   **路由注册顺序最终为：**
   1. `app.include_router(chat_router)` — /v1/chat/completions
   2. `app.include_router(responses_router)` — /v1/responses
   3. `app.post("/responses")` — /responses
   4. `app.include_router(admin_router)` — /api/admin/*
   5. `app.get("/health")` — /health
   6. `app.mount("/assets", ...)` — /assets/*
   7. `app.get("/{full_path:path}")` — catch-all

   注意：
   - `app.mount` 的 StaticFiles 必须只挂载 `/assets` 子目录，不要挂载根目录 `/`，否则会拦截所有请求
   - catch-all 路由用 `@app.get("/{full_path:path}")`，不用 mount
   - 当 static/index.html 不存在时（纯 API 模式），catch-all 返回 JSON 消息
